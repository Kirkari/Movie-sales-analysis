 ## Project Overview

The movie industry generates massive amounts of data related to movie releases, genres, ratings, audience votes, and box office revenue. However, raw data alone does not provide clear insights for decision-making.
This project uses Power BI to transform raw IMDb movie data into interactive dashboards and meaningful visual insights that help identify trends, performance patterns, and key success factors in the movie industry.

## Objective

To analyze IMDb movie data and provide data-driven insights on:
Movie performance over time
Genre popularity and financial success
Audience engagement and ratings impact
Top movies and directors

## Tools & Technologies

Power BI – Data modeling, visualization, dashboards
DAX – Measures and calculated insights
Data Cleaning & Transformation – Power Query
